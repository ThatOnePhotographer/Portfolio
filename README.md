# Portfolio

A simple static portfolio website with:

- Home page featuring a gallery section for images
- About page for your story and skills
- Contact page with an email contact form

## How to use

1. Open `index.html` in a browser.
2. Add your own images to the `images/` folder.
3. Update the image `src` values in `index.html` to your image file names.
4. Customize your name, bio, and email in `about.html` and `contact.html`.

## Files

- `index.html` — main portfolio landing page
- `about.html` — about me page
- `contact.html` — contact page with a form
- `styles.css` — site styling
- `scripts.js` — contact form mailto handler
- `images/placeholder.svg` — default image placeholder
